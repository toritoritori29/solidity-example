# セットアップ
1. 検証用にローカルイーサリウムネットワーク(ganache)の構築が必要です.公式ホームページからダウンロードしてください.
https://trufflesuite.com/ganache/


# 新しいexampleを作るとき
``` bash
$ mkdir exampleX
$ cd exampleX
$ npx truffle init
```
